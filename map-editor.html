Map Editor Help

Tools

  • V - Select tool: Click to select territories
  • P - Polygon tool: Click to add points, double-click to finish
  • R - Rectangle tool: Click and drag to draw
  • C - Connect tool: Click two territories to connect them

Navigation

  • Space + Drag - Pan the view
  • + / - - Zoom in/out
  • Mouse wheel - Zoom in/out
  • 0 - Reset zoom to 100%

Editing

  • Ctrl+Z - Undo
  • Ctrl+Y - Redo
  • Delete - Delete selected territory
  • Ctrl+D - Duplicate selected territory
  • Escape - Cancel current operation

File Operations

  • Ctrl+N - New map
  • Ctrl+O - Load map
  • Ctrl+S - Export map

Tips

  • Use the grid for precise alignment
  • Import a reference image to trace over
  • Save frequently while working
  • Connect territories before exporting
